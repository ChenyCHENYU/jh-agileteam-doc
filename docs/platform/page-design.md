# 页面设计

> 页面设计器是低代码平台的核心可视化工具，通过拖拽组件、配置属性、绑定数据源来构建页面。

---

页面设计是平台自研的低代码的核心功能之一，提供了功能强大的页面设计器，通过拖拽方式生成业务画面，绑定好接口及参数后即可完成业务功能。JH4J-CODE平台页面设计与常规低代码页面设计不同，它串联了平台的接口管理模块，进一步提升低代码灵活性。目前支持50种业务组件，支持样式自定义编辑、数据编辑、动态渲染、事件监听、实时预览、快速模板、高级属性、大纲树等。页面设计器不限定页面类型，不仅支持表单类的业务画面，也支持门户页等自定义任意样式组合风格的页面。

页面组件

页面组件可以理解为一个Vue组件或者一个Vue文件，它包含：template对应画布区域内容、script对应代码编辑器的组件选项。

通过开发人员预定义的页面组件简化应用程序开发过程，无需拖拽组件编写组件间逻辑，大大减少人力成本。

平台组件

平台组件是组件库中平台提供的组件，包括基本组件和高级组件。将平台组件拖拽到画布区域后，运行后称之为运行时组件实例。PC端平台组件名以Component结尾，如：ButtonComponent；移动端组件以Mobile开头，如MobileButton。组件提供了属性配置、事件监听等可配置功能；通过$refs可以获取组件运行实例，可以调用组件的方法、属性，具体组件有哪些方法和属性，可以点击组件文档查看。

通过提供预定义的组件和可视化操作界面开发应用，降低开发难度，提高开发效率

### 组件变量

组件变量是组件定义的可以提供给组件内拖拽的平台组件使用的变量，等同于Vue中data或setup定义的变量，可以通过this.访问到。组件变量分为：接口变量、自定义变量、组件数据、组件属性、上下文变量。

通过组件变量，可以实现用户界面元素的动态更新，响应用户操作或外部数据变化。

开发人员开发阶段

组件变量减少了数据管理和传递的复杂性，使开发人员可以更快地构建和迭代应用程序。
