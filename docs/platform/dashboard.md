# 可视化大屏

> 系统管理 → 低代码 → 大屏设计

可视化大屏用于构建数据可视化展示页面，支持拖拽组件、配置数据源、实时预览。

---

## 组件库

大屏专用组件库：

| 类型 | 组件 |
|------|------|
| 图表 | 折线图、柱状图、饼图、雷达图、散点图 |
| 指标 | 数字翻牌器、仪表盘、水球图 |
| 媒体 | 图片、视频、轮播 |
| 装饰 | 边框、装饰线、标题、背景 |
| 交互 | 时间选择器、下拉筛选器、按钮 |

---

## 画布

- 支持自定义画布尺寸（默认 1920×1080）
- 自由拖拽定位组件
- 支持组件分组、锁定、隐藏

---

## 配置

| 配置 | 说明 |
|------|------|
| 组件样式 | 颜色、字体、边框、动画 |
| 组件数据 | 绑定数据源、配置接口参数 |
| 交互事件 | 点击事件、联动筛选 |
| 自适应 | 画布缩放模式（等比缩放/全屏铺满） |

---

## 数据流向过程

```text
数据源（接口/模型）
  ↓ 数据格式化（字段映射）
  ↓ 组件绑定（图表/指标）
  ↓ 实时渲染
  ↓ 交互联动（筛选器 → 重新请求 → 刷新图表）
```
