# 可视化大屏

> 可视化大屏用于构建数据可视化展示页面。

---

低代码BI大屏已经成为数据可视化领域的一项重要技术，为用户提供了一种交互式、生动的数据展示方式，低代码BI大屏内置了丰富的组件和多种数据源，组件可以通过自由布局的形式在画布中进行展示，并根据不同的属性配置来控制组件的展示形式。

通过对组件自由布局的形式，来完成可视化页面的配置

包含数据源，图片上传，数据管理，撤销回退等操作功能

平台用户

平台用户

组件库

概念

BI大屏设计的核心在于构建各种各样的组件，这些组件可以是图表、文本、图片、视频等等，用于呈现不同类型的数据。

设计原则

多样性

组件库应包括多种类型的组件，以满足不同的数据展示需求。同时，这些组件应具有高度的可定制性，用户可以根据自己的需求对组件进行样式和数据的定制。这有助于创造多样性的大屏设计，使之更具吸引力。

可复用性

组件库中的组件应该是可复用的，这意味着它们可以在不同的大屏设计项目中重复使用。这有助于提高设计的效率和一致性，同时也降低了维护的成本。

响应式设计

大屏设计通常需要在不同的屏幕尺寸和设备上进行展示，因此组件库应该支持响应式设计。这意味着组件应该能够自动适应不同的屏幕大小和设备类型，以确保在各种环境下都能提供良好的用户体验。

可扩展性

组件库的可扩展性是确保其长期可用性和适应性的关键。可扩展性意味着组件库应该允许开发人员根据需要添加新的自定义组件或扩展现有组件的功能。

画布

概念

画布是BI大屏设计的基础，它是组件的容器，决定了组件的布局和排列方式。

设计原则

自由布局

画布应该支持自由布局，允许用户将组件自由拖拽、缩放和旋转，以实现个性化的设计布局。这种自由度可以增加用户的创造力和灵活性

对齐和分布工具

画布应该提供对齐和分布工具，帮助用户在设计中保持组件的对齐和均匀分布。这有助于提高设计的美观度和一致性。

分层管理

大屏设计可能包含大量的组件，画布应该提供分层管理功能，允许用户轻松地调整组件的层级顺序，确保重要的信息始终处于视觉上的前景。

画布背景

画布的背景可以是简单的白色或其他单色背景，也可以是图像、地图或其他信息。背景的选择会影响整体的设计感觉和可视化效果，因此应根据具体的设计目标来选择。

配置

概念

属性配置决定了组件的外观和行为。

设计原则

直观性界面

属性配置界面应该是直观的，设计师和非技术用户能够轻松地理解和使用。采用图形化界面、拖放操作、即时预览等技术，以提高用户友好度。

数据绑定

属性配置应支持数据绑定，允许用户将组件的属性与数据源关联起来。这样，组件可以实时更新以反映数据的变化，提供更具实用性和决策支持的可视化。

预览和撤销

属性配置界面应提供预览功能，让用户实时查看属性的变化效果。同时，还应该提供撤销和重做功能，以便用户可以随时恢复到之前的配置状态，减少误操作的影响。

选项和设置

属性配置应提供丰富的选项和设置，以满足不同用户的需求。这包括颜色选择器、字体选择、数据过滤条件、数据排序方式等。多样性的选项可以增加可视化的定制性。

数据流向过程

数据流向的起点是数据源。数据源可以是各种类型的，包括静态数据、API数据、模型数据等。这些数据源存储了待展示的信息和数据。一旦数据源被确定，前端需要通过合适的手段从数据源中提取数据，并对提取的数据进行处理，处理完成后将数据传递到前端大屏的组件中，组件会使用数据来生成可视化内容，这一过程中数据一直保持单向流动。

### 新建大屏

点击新建大屏进入新建页面，在新建页面提供了丰富模板页面进行选择，也可以选择空白模板进行创建，之后需要填写分组，大屏名称，屏幕尺寸进行页面的创建。

![分类管理](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/dashboard/02.png)

### 分类管理

在分类管理页面可以新建模板，把不同模板进行分类展示，便于在新建大屏的时候进行选择。

![图片上传](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/dashboard/03.png)

### 图片上传

平台提供图片上传的功能，可以通过图片组件将所需要的图片进行上传展示。

![菜单功能](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/dashboard/04.png)

### 菜单功能

每一个组件通过鼠标右击的形式，展示出组件菜单，组件菜单中包含锁定，组合，删除，复制各种快捷的操作。

![导航栏](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/dashboard/05.png)

### 导航栏

最上方的导航栏依次提供返回主页，保存，导出图片，预览，撤销，重做的一些能力。

![图层](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/dashboard/06.png)

### 图层

拖入画布中的组件会在左侧的图层中展示组件的信息，鼠标右击将展示图层的一些常用的功能，比如锁定，组合，删除，复制，置顶，置底，上移，下移。
