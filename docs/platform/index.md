# 低代码平台用户操作手册

> FSI2 低代码平台 V3.1.0 · 发布日期 2026-02-26 · 江苏金恒信息科技股份有限公司

低代码平台是一种以可视化方式进行应用程序开发的工具，通过可拖拽的组件、预置的模块和自动生成的代码，快速构建功能完善的应用程序。

::: tip 关于截图
本手册由 Word 文档转换而来，部分操作截图以 `[截图]` 标注。如需查看完整截图版，请参考原始 Word 手册。
:::
