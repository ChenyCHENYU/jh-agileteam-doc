# 文件服务

> 系统管理 → 文件服务


![文件管理](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/file-service/01.png)
文件服务提供文件索引管理和文件管理能力。

---

## 索引管理

> 系统管理 → 文件服务 → 索引管理

管理文件存储索引，配置文件存储位置（本地/MinIO/阿里云 OSS 等）。

### 操作说明

| 操作 | 说明 |
|------|------|
| 新增索引 | 配置存储类型、路径、访问权限 |
| 编辑索引 | 修改存储配置 |
| 删除索引 | 删除索引配置（不影响已存储文件） |

---

## 文件管理

> 系统管理 → 文件服务 → 文件管理

管理平台中上传的文件。

### 操作说明

| 操作 | 说明 |
|------|------|
| 文件列表 | 按文件名、类型、上传时间筛选 |
| 上传文件 | 支持拖拽上传、多文件上传 |
| 预览/下载 | 在线预览图片/文档，下载文件 |
| 删除文件 | 删除文件（同时清理索引） |
