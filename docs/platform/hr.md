# 人事管理

> 系统管理 → 人事管理

人事管理包括用户管理、部门管理、公司管理、用户组管理、岗位管理和个人中心。

---

## 用户管理

> 系统管理 → 人事管理 → 用户管理

管理平台系统用户，包括新增、编辑、删除、启用/停用、重置密码等操作。

### 关键字段

| 字段 | 说明 |
|------|------|
| 账号 | 登录账号 |
| 姓名 | 用户真实姓名 |
| 手机号 | 用于绑定钉钉等第三方账号 |
| 工号 | 员工编号 |
| 部门 | 所属部门 |
| 公司 | 所属公司 |
| 岗位 | 所属岗位 |
| 角色 | 分配的角色 |
| 状态 | 启用/停用 |

---

## 部门管理

> 系统管理 → 人事管理 → 部门管理

管理组织架构中的部门树形结构，支持新增、编辑、删除、排序。

---

## 公司管理

> 系统管理 → 人事管理 → 公司管理

管理平台中的公司信息，每个公司可关联多个部门和用户。

---

## 用户组管理

> 系统管理 → 人事管理 → 用户组管理

将用户按业务场景分组，便于批量授权。

---

## 岗位管理

> 系统管理 → 人事管理 → 岗位管理

管理平台中的岗位信息。

---

## 个人中心

> 日常办公 → 个人中心

普通用户可查看和修改个人信息、修改密码、切换公司等。

### 功能

| 功能 | 说明 |
|------|------|
| 基本信息查看 | 查看个人信息 |
| 修改密码 | 修改登录密码 |
| 公司切换 | 切换当前公司上下文 |
| 主题切换 | 切换个性化主题 |
