# 人事管理

> 人事管理包括用户管理、部门管理、公司管理、用户组管理、岗位管理和个人中心。

---

包含用户管理、部门管理、公司管理和用户组管理及岗位管理模块。

用户管理

用户是系统使用者用来登录系统的媒介，具有用户名、密码、所属部门等属性。用户管理是系统后台对系统使用的用户进行管理的功能模块。

管理使用系统的用户

用户增删改查、用户密码管理、用户权限分配时

提供对用户增删改查的基本功能。以及对用户密码管理和用户权限分配的功能。

管理员用户进入“业务管理”子系统，在“人事管理”目录下打开“用户管理”页面。

新增用户

用户管理页面如下图所示。

![点击新增，录入用户登录相关的信息，保存后用户新增成功。](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/01.png)

点击新增，录入用户登录相关的信息，保存后用户新增成功。

![重置密码](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/02.png)

### 重置密码

点击“重置密码”，确认后密码重置成功。

![角色分配](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/03.png)

### 角色分配

为用户分配系统角色的功能 不同的角色看到的菜单权限不同。点击“分配角色”，弹出分配角色弹框，如下图所示：

![在弹框左侧有两个tab页，第一个“角色”tab页可通过点击新增直接给用户分配角色](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/04.png)

在弹框左侧有两个tab页，第一个“角色”tab页可通过点击新增直接给用户分配角色并设置角色有效期，有效期为空则表示永久有效，“删除”操作将此用户已分配角色去除。

![第二个“用户”tab页可通过选择系统中存在的用户进行角色复制并设置角色有效期，有](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/05.png)

第二个“用户”tab页可通过选择系统中存在的用户进行角色复制并设置角色有效期，有效期为空则表示永久有效，“删除”操作将此用户已分配角色去除。

删除用户

管理员删除用户之后，此用户在系统中不能登录。点击“删除”，弹出确认删除弹框，点击确认后用户删除。

![停用用户](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/06.png)

### 停用用户

点击更多，点击停用，进行用户的停用，停用后的用户将无法登录系统或被用户挑选框选中。

![一键增加用户默认角色](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/07.png)

### 部门管理

提供对使用系统的公司组织架构管理的功能，整体架构包含公司子公司、部门、下级部门。

用户所在部门增删改查

管理员用户进入“业务管理”子系统，在“人事管理”目录下打开“部门管理”页面。

新建部门

部门管理页面如下图所示：

在对应的层级的某个节点下点击新增按钮，新增的是子级别的部门。点击确定，完成部门新增。可根据部门排序字段设置排序展示。每个部门都可维护属于自己的岗位，部门人员的所属岗位信息在部门岗位范围内进行挑选分配。

![部门删除](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/hr/12.png)

### 部门删除

### 用户组管理

管理系统用户组并为用户组统一配置权限

管理系统用户分组，实现权限的批量配置与高效继承

新建用户组、为用户组分配权限、维护用户组成员时

1. 提供用户组的增删改查功能；

2. 支持为用户组赋予 / 回收系统权限；

3. 支持将用户批量加入 / 移出用户组

管理员用户进入“业务管理”子系统，在“人事管理”目录下打开“用户组管理”页面。

新建用户组

点击新增按钮，弹出添加弹框，录入用户组相关信息，并在下方表格中添加用户组成员，维护完成后点击确认按钮。
