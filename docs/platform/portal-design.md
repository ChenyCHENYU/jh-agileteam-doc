# 门户页设计

> 系统管理 → 低代码 → 门户页设计

门户页设计用于构建系统的首页门户，包括模板市场、门户组件和门户自定义。

---

## 模板市场

提供预置门户页模板，可一键应用到当前子系统。

### 操作说明

1. 进入模板市场，浏览可用模板
2. 选择模板，预览效果
3. 点击应用，将模板配置应用到当前门户页

---

## 门户组件

门户页专用组件，与页面设计的基础/高级组件不同，面向门户首页场景：

| 组件 | 说明 |
|------|------|
| 数据卡片 | 展示 KPI 指标 |
| 图表组件 | 展示趋势图/饼图/柱状图 |
| 快捷入口 | 快捷功能入口 |
| 待办列表 | 展示待办事项 |
| 公告通知 | 展示系统公告 |
| 轮播图 | 图片轮播展示 |

---

## 门户自定义

在模板基础上自定义门户页布局和内容。

### 操作说明

1. 选择基础模板或从空白开始
2. 拖拽门户组件到画布
3. 配置组件数据源和样式
4. 预览并保存发布
