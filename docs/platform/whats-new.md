# 新特性（V3.1.0）

> FSI2 低代码平台 V3.1.0 版本新增功能。

---

## 防重提交

防止用户重复提交表单数据。对同一接口的重复请求在短时间内自动拦截，避免产生重复数据。

---

## 操作日志

增强的操作日志能力，记录用户在平台中的关键操作行为，支持按操作类型、用户、时间范围查询和导出。

---

## 高代码数据权限

为高代码（手写代码）开发场景提供数据权限支持。通过注解或配置方式，让非低代码生成的接口也能使用平台的数据权限规则。

---

## 熔断限流

平台级熔断限流能力，保护核心服务在高并发场景下的稳定性：

| 能力 | 说明 |
|------|------|
| 接口限流 | 按接口维度配置 QPS 上限 |
| 服务熔断 | 服务异常时自动熔断，快速失败 |
| 降级策略 | 熔断后返回降级数据或友好提示 |
| 恢复探测 | 熔断后自动探测服务恢复，自动恢复 |
