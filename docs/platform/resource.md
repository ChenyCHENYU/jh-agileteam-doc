# 资源管理

> 系统管理 → 资源管理


![图片库管理](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/09.png)
资源管理包括图标库管理和图片库管理。

---

## 图标库管理

> 系统管理 → 资源管理 → 图标库

管理平台中使用的 SVG 图标资源。

### 操作说明

| 操作 | 说明 |
|------|------|
| 新增图标 | 上传 SVG 文件，填写图标名称和分类 |
| 图标分类 | 按业务分类管理（系统图标、业务图标等） |
| 使用图标 | 在菜单配置、按钮配置等位置选择图标 |
| 删除图标 | 删除未被使用的图标 |

---

## 图片库管理

> 系统管理 → 资源管理 → 图片库

管理平台中使用的图片资源（Logo、背景图、业务图片等）。

### 操作说明

| 操作 | 说明 |
|------|------|

![上传图片](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/13.png)
![图片详情](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/14.png)

| 新增图片 | 上传图片，填写名称和分类 |
| 图片分类 | 按用途分类管理 |
| 预览/下载 | 在线预览和下载 |

![删除图片](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/15.png)
| 删除图片 | 删除未被使用的图片 |
