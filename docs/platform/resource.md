# 资源管理

> 资源管理包括图标库管理和图片库管理。

---

系统中所有图标和静态图片资源在资源管理中管理。

图标库管理

管理图标

提供图标的上传、删除、分组功能

系统管理〉静态资源〉资源管理，图标库管理包括图标分组、图标管理，图标分组用于对图标进行分组管理。

![新增分组](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/01.png)

### 新增分组

### 点击上传按钮

点击新增分组，如下图所示：

![输入分组名称，点击确定完成分组创建](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/02.png)

输入分组名称，点击确定完成分组创建

删除分组

点击删除按钮，确定完成删除，如下图所示：

![修改分组名称](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/03.png)

### 修改分组名称

点击先改名称按钮，输入新的分组名称，点击确定：如下图所示；

![上传图标](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/04.png)

### 上传图标

点击上传按钮，选择svg文件，文件命名规范为：pccom-EchartsHorizontalBar横向柱状图。svg，竖线前面为图标编码，后面为图标名称。

删除图标

点击删除按钮，完成删除

![批量删除图标](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/05.png)

### 批量删除图标

选中图标，点击批量删除，完成删除

![移动分组](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/06.png)

### 移动分组

点击移动按钮，完成分组移动

![批量移动分组](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/07.png)

### 批量移动分组

选择图标后，点击批量移动按钮

![修改图标信息](https://cheny-chenyu.oss-cn-chengdu.aliyuncs.com/jh/platform/resource/08.png)

### 修改图标信息

### 点击修改按钮，完成修改
