# AI 最佳实践

<AuthorTag :authors="['ZhuXiang','CHENY']" />

## 概述

AI 最佳实践按照**能力层级**划分为四个等级，帮助团队循序渐进地掌握 AI 辅助开发能力。

## 层级体系

| 层级 | 名称 | 说明 |
|---|---|---|
| **L1** | 提示词 | 基础提示词工程，掌握与 AI 高效对话的方法论 |
| **L2** | Skill | 单个 AI Skill 的编写与使用，完成特定开发任务 |
| **L3** | Skills & MCP | 多 Skill 编排 + MCP 协议集成，实现复杂工作流 |
| **L4** | CLI | 命令行工具链，将 AI 能力融入自动化流水线 |
