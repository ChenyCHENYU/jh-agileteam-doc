# ⑦ 规范审计

**Skill 名称**：`convention-audit`

**触发词**：`审计规范`、`代码检查`

**规则文件**：`.github/skills/convention-audit/skills.md`

---

## 功能

对生成或手写的代码进行全面规范审计，确保交付质量。

---

## 审计清单

| 审计项 | 检查内容 |
|---|---|
| 文件结构 | 三文件分离（`.vue` 无 `<style>` 块） |
| 命名规范 | BEM / 路由 name / API 命名约定 |
| 样式令牌 | 禁止硬编码颜色 / 圆角 / 阴影 |
| 类型安全 | `pnpm type-check` 零错误 |
| 组件使用 | 全局组件 `C_` 正确引用 |
| Mock 配套 | 每个 API 有对应 Mock |

---

## 审计流程

```
生成 / 修改代码
  → P0 问题静默修复（类型错误、未使用变量）
  → P1 问题修复 + 报告
  → pnpm type-check（零错误）
  → 交付
```
