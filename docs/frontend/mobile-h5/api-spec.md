# ② 接口规格

**Skill 名称**：`api-spec`

**触发词**：`生成接口规格`、`接口字段说明`

**规则文件**：`.github/skills/api-spec/skills.md`

---

## 功能

根据原型分析结果，生成前后端对齐的接口字段规格文档，包含请求参数、响应结构、字段类型与校验规则。

---

## 输出内容

- **请求参数表** — 参数名、类型、必填、说明
- **响应结构表** — 字段名、类型、说明
- **枚举值映射** — 状态码、类型码的含义说明
